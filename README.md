# egg-framework-zxh



## QuickStart

```bash
$ npm install
$ npm test
```

publish your framework to npm, then change app's dependencies:

```js
// {app_root}/index.js
require('egg-framework-zxh').startCluster({
  baseDir: __dirname,
  // port: 7001, // default to 7001
});

```
